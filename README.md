# Визуализатор и конструктор нейронных сетей с графическим интерфейсом.

---

### Интерфейс программы

![GUI скриншот](images/gui.png)

## Возможности

- Создание нейросети с настройкой слоев и нейронов
- Обучение с отображением ошибок в реальном времени
- Прогнозирование результатов
- Сохранение и загрузка обученных сетей
- Визуализация весов и смещений

---

```bash

Проект построен на паттерне MVC (Model-View-Controller)

Структура проекта:
main.py Точка входа в приложение
view.py Графический интерфейс (View)
neural_network.py Логика нейросети (Controller)
config.py Настройки и конфигурация
node.py Визуализация нейрона
arrow.py Визуализация связи (веса)
value_box.py Визуализация блока значений
generate_features.py Генератор признаков для тестирования
generate_labels.py Генератор меток для тестирования
requirements.txt Зависимости проекта

1. Установка зависимостей

pip install -r requirements.txt

2. Запуск приложения

python main.py

3. Руководство по использованию

3.1 Создание нейросети
3.1.1 В левой панели выберите количество нейронов для каждого слоя
3.1.2 Нажмите кнопку "Создать нейросеть"
3.1.3 На центральном холсте появится визуализация сети

3.2 Обучение нейросети
3.2.1 Нажмите "Загрузить хар-ки НС" — выберите CSV файл с признаками
3.2.2 Нажмите "Загрузить метки НС" — выберите CSV файл с метками
3.2.3 Настройте параметры обучения (скорость, эпохи, частота обновления)
3.2.4 Нажмите "Начать обучение"

3.3 Прогнозирование
3.3.1 Нажмите "Загрузить ф-цию прогноз." — выберите тестовые признаки
3.3.2 Нажмите "Начать прогнозирование"

Результаты появятся в правой нижней панели

Генерация тестовых данных
python generate_features.py  Создает признаки (3 класса по 700 примеров)
python generate_labels.py Создает one-hot метки

Они создадут следующие файлы:
train_features.txt (обучающие признаки)
test_features.txt (тестовые признаки)
train_labels.txt (обучающие метки)
test_labels.txt (тестовые метки)
